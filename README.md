# 🚀 Codemaster

## 📌 Sobre o Projeto

O **Codemaster** é um site de portfólio desenvolvido utilizando **HTML, CSS e JavaScript**.
A estrutura do site está completa e organizada, com diferentes seções e layout responsivo. Algumas funcionalidades foram implementadas para melhorar a experiência do usuário.

---

## 🏗 Estrutura do Projeto

O site possui as seguintes partes principais:

* Página inicial (Home)
* Seção **Sobre Mim**
* Seções de conteúdo organizadas
* Layout responsivo para diferentes dispositivos
* Estrutura semântica em **HTML**
* Organização de **estilos (CSS)** e **scripts (JavaScript)**

---
