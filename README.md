# 🚀 CodeMaster Dev - Portfólio Interativo

## 📌 Estado Inicial do Projeto

Recebi a estrutura inicial do projeto contendo os arquivos **HTML**, **CSS**, imagens e um arquivo **JavaScript vazio**.

O layout do portfólio já está funcionando visualmente, porém ainda não possui interatividade.
Atualmente:

* O **menu mobile não abre** ao clicar no botão hambúrguer.
* Os **links de navegação não destacam a seção ativa**.
* O **formulário recarrega a página ao ser enviado**.
* Não existem **animações ao rolar a página**.

O objetivo deste projeto é utilizar **JavaScript Vanilla** para adicionar interatividade e comportamento dinâmico ao site.

---

# 🎯 Objetivo do Projeto

Transformar um site estático em um **portfólio interativo**, aplicando conceitos fundamentais de JavaScript e manipulação do DOM.

Durante o desenvolvimento serão aplicadas técnicas como:

* Seleção e manipulação de elementos do **DOM**
* Uso de **eventos**
* Criação de **funções**
* Manipulação de **classes CSS**
* Integração com **API usando Fetch**
* Detecção de **scroll na página**

---

# 🛠 Funcionalidades que serão implementadas

### 📱 1. Menu Mobile

Criar um menu responsivo que abre e fecha ao clicar no botão hambúrguer, utilizando manipulação de classes com JavaScript.

### 🔗 2. Links de Navegação Ativos

Destacar automaticamente o link da seção que o usuário está visualizando durante a rolagem da página.

### 📩 3. Envio de Formulário com API

Interceptar o envio do formulário utilizando `preventDefault()` e enviar os dados para uma API utilizando **Fetch**, sem recarregar a página.

### ✨ 4. Animação ao Scroll

Adicionar animações de entrada em elementos da página quando eles aparecem na tela durante o scroll.
